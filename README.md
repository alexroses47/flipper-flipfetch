# FlipFetch

Sistema de información estilo fastfetch para Flipper Zero.

<img width="512" height="256" alt="Screenshot-20260327-213438" src="https://github.com/user-attachments/assets/7496a6b4-501f-4fa9-a866-bc523bcc6628" />


## Muestra
- Firmware version
- Build date  
- Batería % y voltaje
- RAM libre
- SD libre
- Uptime

## Instalación
Descarga el `.fap` de Releases y cópialo a `/apps/Tools/` en tu SD.

## Compilar
```bash
ufbt
```

## Compatibilidad
Probado en Momentum firmware mntm-012.
